# 050_bell asset contract

`050_bell` has two official instances (`base0` and `base1`) and a functional
contact point used by `click_bell.check_success()`. Bounded instance probes may
select only these existing ids. The task must keep the actor name, static
collision behavior, functional-point semantics, and description metadata.

The Offline Extractor fingerprints the small model/description JSON files. It
does not copy, embed, or hash the multi-megabyte mesh files into prompts.
